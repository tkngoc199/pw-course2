1. Giải thích các câu lệnh
- git init: tạo ra thư mục .git trong project. Thư mục .git chứa toàn bộ dữ liệu Git: history, cấu trúc, object,..
- git add <file_name>: thêm file cụ thể từ working directory vào vùng staging
- git add . : thêm tất cả các file từ working directory vào vùng staging
- git commit -m "message": đóng gói toàn bộ file trong vùng staging, tạo ra một snapshot mới và lưu vào vùng repository kèm theo thông điệp mô tả
